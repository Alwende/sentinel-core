# Sentinel-Core v1.0
**Lead Architect:** alwende

## Overview
Sentinel-Core is a lightweight, zero-dependency system auditor designed for Linux environments. 

## Installation
1. Clone the repository.
2. `chmod +x sentinel_audit.sh`
3. `./sentinel_audit.sh`

## Features
- Real-time Disk Utilization Audit
- Human-Readable Memory Analysis
- Security Check (Active User Sessions)
