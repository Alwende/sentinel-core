# Project Charter: Sentinel-Core
**Project ID:** P-17-AUDIT  
**Lead Architect:** Alwende  

## 1. Vision Statement
To provide Linux systems administrators with a zero-dependency, high-velocity audit tool that ensures node health and security compliance in enterprise environments.

## 2. Scope
- **In-Scope:** Disk monitoring, Memory utilization, Active session auditing, and Log persistence.
- **Out-of-Scope:** External network penetration testing and GUI development (v1.0).

## 3. Success Metrics
- **Performance:** Execution time < 2 seconds.
- **Accuracy:** 100% parity with standard Linux binaries (`df`, `free`, `who`).
- **Adoption:** Open-source availability via GitHub for global community use.
